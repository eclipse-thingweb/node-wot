Smart Coffee Machine

This is a simulated smart coffee machine with a range of capabilities. It is exposed as a virtual Thing to demonstrate the capabilities of Web of Things and, particularly, the node-wot runtime environment.

More details on the coffee machine usage can be found at http://www.thingweb.io/smart-coffee-machine.html.

The Thing Description for the coffee machine is available at http://plugfest.thingweb.io:8083/smart-coffee-machine.
